After covering the main aspects of the Joget solution and installing it on your system, it's time to create an application. This section presents the essential phases for creating your first application on Joget.

We will design a Traves Request app to illustrate this guide. Each page presents the basic steps to create each part of this application. Follow the steps below to have the complete application running in just a few minutes:

- **Step 1**: [Design a new app](./create-an-app/design-new-app).
- **Step 2**: [Define a form](./create-an-app/design-a-form).
- **Step 3**: [Create a list](./create-an-app/design-a-list).
- **Step 4**: [Design a new UI](./create-an-app/design-a-ui).
- **Step 5**: [Stabilish a process](./create-an-app/design-a-process).
