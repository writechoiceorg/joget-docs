To continue the development of our Traves Request app, we will create a form. Follow the steps on this page to create your first form.

- **Step 1**: Access the **app design** page. If you just designed your first app, you're aready on the app design page.
- **Step 2**: Create a new form using the **+** button.
- **Step 3**: Now, you will enter the form details, including:

<table>
  <thead>
    <tr>
      <th>Field</th>
      <th>Description</th>
      <th>Value for this example</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Form ID</td>
      <td>A unique identifier.</td>
      <td>travelRequestForm</td>
    </tr>
    <tr>
      <td>Form Name</td>
      <td>A unique descriptive name.</td>
      <td>Travel Request Form</td>
    </tr>
    <tr>
      <td>Table Name</td>
      <td>The database table name to store the form data.</td>
      <td>travel_requests</td>
    </tr>
    <tr>
      <td>Description</td>
      <td>Describes the form functionalities and purpose.</td>
      <td>Travel request form used as example on the app creation process.</td>
    </tr>
  </tbody>
</table>

:::info Forms sharing information
Different forms can share data by pointing to the same table name. You can use this functionality to create comprehensive tables to be used as database.
:::

- **Step 4**: Click **Save** to create the form, as a result, the Form Builder will open. If your browser blocks the popup, click on the form name to launch it.

**[CONTINUE FROM HERE]**

- **Step 5**: Customize the form

  Click Edit Section to rename the section label.
  Add form elements as needed.
  Rename IDs and field labels.
  Configure each element's properties.

Step 6: Save Your Work

After customizing the form, remember to save it.
Step 7: Done!

You've successfully designed a form. Stay tuned for Part 3 where we'll explore designing datalists.
