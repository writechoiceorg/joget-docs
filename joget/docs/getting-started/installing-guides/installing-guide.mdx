You can also download Joget from our website. 
This guide will walk you through the process of downloading, installing and starting to use Joget step-by-step.

Below you will find all steps to install and initialize Joget, and a video with this same walkthrough.

### Step 1 - Download Joget installer

1. Access our [Download](https://www.joget.org/download/) page.

<Images
  imgURL="img/getting-started/install-guides/download/download_options.png"
  alt="joget download options"
/>

2. Choose an **Edition**. [Here](https://www.joget.org/product/features/) you can learn more about each option.
3. Select the download that fits your operating system.
4. Download it.

### Step 2 - Installing

1. Execute the dowloaded file from Step 1.
2. You will be presented with our installation setup.

<Images
  imgURL="img/getting-started/install-guides/download/install_start.png"
  alt="installation start"
/>

3. Click **Next**.
4. **Read** and **agree** to our License Agreement.

<Images
  imgURL="img/getting-started/install-guides/download/read_agree.png"
  alt="read and agree terms"
/>

5. Choose which components you desire to install. We fully recommend installing all selected components.

<Images
  imgURL="img/getting-started/install-guides/download/components_selection.png"
  alt=""
/>

6. Click **Next**.
7. Select your **Destination Folder**.

<Images
  imgURL="img/getting-started/install-guides/download/browse_folder.png"
  alt="installation folder selection"
/>

8. Click **Install**.

<Images
  imgURL="img/getting-started/install-guides/download/install_click.png"
  alt="installation"
/>

9. Click **Finish**.

### Step 3 - First starting Joget

1. Open the **Destination Folder** you selected at Step 2.7.
2. Find the **Start Joget Server** shortcut, and double click it.

<Images
  imgURL="img/getting-started/install-guides/download/joget_shortcut.png"
  alt="joget shortcut"
/>

Here our program will open two instances of your systems terminal, one for MariaDB and the second to run Joget.

3. The Application's homepage will be accessible when the terminal running Joget shows a message that the server has started.
When this happens, you can access [localhost:8080/jw](http://localhost:8080/jw) to open the App Center.

<Images
  imgURL="img/getting-started/install-guides/download/localhost8080jw.png"
  alt="application running on localhost8080"
/>

### Step 4 - Create an App

- After installing our application, you can proceed to the [Pratform Overview](/docs/getting-started/platform-overview) to learn about the main existing elements available on the Joget applications.
