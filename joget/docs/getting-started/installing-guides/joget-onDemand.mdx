**Joget DX On-Demand** is a hosted Joget service provided to help you to get started right away without the fuss of setting up things yourself. Below you have the steps you need to complete to start with our hosted service.

- **Step 1**: Access [cloud.joget.com](https://cloud.joget.com/jw/web/userview/cloud/u1/_/overview).
- **Step 2**: Click **Sign Up Free** and follow instructions for creating a new account.

  imgURL="img/getting-started/install-guides/signupfree.png"
  alt="Form section creation example"
/>

After installing our extension, you can proceed to the [Pratform Overview](/docs/getting-started/platform-overview) to learn about the main existing elements available on the Joget applications.
