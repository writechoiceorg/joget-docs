El primer paso para crear una nueva aplicación con Joget es diseñarla. Sigue los pasos en esta página para diseñar tu primera aplicación.

- **Paso 1**: Para comenzar, accede al Centro de Aplicaciones de Joget.
- **Paso 2**: Inicia sesión como Administrador utilizando las credenciales de administrador por defecto.
  - **Nombre de usuario**: admin
  - **Contraseña**: admin
- **Paso 3**: Localiza la **Barra de Administración** en la parte inferior y haz clic en **Todas las Aplicaciones**.
- **Paso 4**: Para comenzar a diseñar una nueva aplicación, después de acceder a la ventana de Todas las Aplicaciones, utiliza el botón **Diseñar una Nueva Aplicación**.
- **Paso 5**: Define los detalles de la nueva aplicación, proporcionando:

<table>
  <thead>
    <tr>
      <th>Campo</th>
      <th>Descripción</th>
      <th>Valor para este ejemplo</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>ID de la Aplicación</td>
      <td>Un identificador único.</td>
      <td>solicitudDeViaje</td>
    </tr>
    <tr>
      <td>Nombre de la Aplicación</td>
      <td>Un nombre descriptivo único.</td>
      <td>Solicitud de Viaje</td>
    </tr>
  </tbody>
</table>

- **Paso 6**: Haz clic en **Guardar** para crear tu aplicación. Serás dirigido a la página de diseño de la aplicación.
