Vamos a crear una interfaz de usuario (UI) para continuar el desarrollo de nuestra aplicación de Solicitud de Viaje. Utilizarás listas para mostrar datos a los usuarios. En este caso, la lista mostrará los datos del formulario de Solicitud de Viaje. Sigue los pasos en esta página para crear tu primera lista.

- **Paso 1**: Accede a la página del **compositor de aplicaciones**.
- **Paso 2**: Crea una nueva lista utilizando el botón **+** en la columna del **Constructor de UI**.
- **Paso 3**: Ahora ingresarás los detalles de la UI, que incluyen:

<table>
  <thead>
    <tr>
      <th>Campo</th>
      <th>Descripción</th>
      <th>Valor para este ejemplo</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>ID</td>
      <td>Un identificador único.</td>
      <td>travelRequestUi</td>
    </tr>
    <tr>
      <td>Nombre</td>
      <td>Un nombre descriptivo único.</td>
      <td>Interfaz de Usuario de Solicitud de Viaje</td>
    </tr>
    <tr>
      <td>Descripción</td>
      <td>Describe el propósito de la lista.</td>
      <td>Interfaz de usuario para la aplicación de Solicitud de Viaje.</td>
    </tr>
  </tbody>
</table>

- **Paso 4**: Haz clic en **Guardar** para crear la UI. Como resultado, se lanzará el **Constructor de UI**.
- **Paso 5**: Diseña la UI. Las categorías se pueden editar y reorganizar arrastrándolas. Los elementos de menú de la UI del panel se pueden colocar dentro de las categorías. Al hacer clic en categorías o elementos, se muestran las opciones disponibles. Las acciones como reordenar, eliminar, seleccionar el padre y editar propiedades se pueden realizar en el panel derecho. Las propiedades individuales del elemento de la UI se pueden modificar haciendo clic en ellos. Para este ejemplo, no se requiere ninguna modificación de la UI.
- **Paso 6**: Haz clic en **Vista Previa** para verificar cómo se ve la UI.
- **Paso 7**: Haz clic en **Guardar** cuando termines de crear la UI.

:::info Disponibilidad de la UI
La aplicación debe estar publicada para que un usuario normal acceda a una UI.
:::

- **Paso 8**: Publica la UI. Ve a la página del compositor de aplicaciones y haz clic en el botón **Publicar** en la parte superior derecha. Haz clic en **Aceptar** para confirmar la creación.

Si accedes al centro de aplicaciones, encontrarás la aplicación de Solicitud de Viaje en la página de inicio.
