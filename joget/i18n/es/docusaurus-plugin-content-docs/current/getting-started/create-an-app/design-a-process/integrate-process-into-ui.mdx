Después de definir los componentes del proceso y mapearlos, puedes integrar el proceso en una interfaz de usuario (UI). Sigue los pasos a continuación para integrar el proceso en la UI creada en los pasos anteriores de esta guía.

- **Paso 1**: Accede a la página del **App Composer** y haz clic en "UI existente" para iniciar el UI Builder.

<Images
  imgURL="img/getting-started/create-an-app/design-a-process/integrate-process-into-ui-1.png"
  wSize="65%"
/>

- **Paso 2**: En el UI Builder, arrastra y suelta un elemento de **Ejecutar Proceso** en el menú. El elemento de Ejecutar Proceso permite al usuario iniciar un proceso desde la vista del usuario.

<Images
  imgURL="img/getting-started/create-an-app/design-a-process/integrate-process-into-ui-2.png"
  wSize="65%"
/>

- **Paso 3**: Selecciona el elemento de Ejecutar Proceso y elige el proceso que se ejecutará. Para este ejemplo, selecciona el proceso de Solicitud de Viaje.

<Images
  imgURL="img/getting-started/create-an-app/design-a-process/integrate-process-into-ui-3.png"
  wSize="65%"
/>

- **Paso 4**: Arrastra y suelta un elemento de **Bandeja de Entrada** en el menú. El elemento de Bandeja de Entrada proporciona al usuario una bandeja de entrada para gestionar sus tareas de flujo de trabajo asignadas.

<Images
  imgURL="img/getting-started/create-an-app/design-a-process/integrate-process-into-ui-4.png"
  wSize="65%"
/>

- **Paso 5**: Haz clic en **Guardar** en la esquina superior derecha para guardar la UI.

¡Felicidades por integrar el proceso en una interfaz de usuario! Ahora puedes publicar la aplicación y ser capaz de ejecutar el proceso y realizar tareas asignadas.
