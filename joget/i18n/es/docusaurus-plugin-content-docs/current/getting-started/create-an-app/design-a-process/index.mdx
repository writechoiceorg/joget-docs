El Proceso es la solución de Joget que definirá el flujo de datos dentro de tu aplicación. Al crear el proceso, definirás condiciones y acciones que se ejecutarán en función de las interacciones del usuario.

Dado que el Proceso es una herramienta más compleja que las anteriores, su presentación se dividirá en pequeñas partes. Utiliza los siguientes enlaces para navegar por la guía de creación del proceso:

- **Paso 1**: [Constructor de Procesos](./design-a-process/process-builder).
- **Paso 2**: [Agregar componentes al proceso](./design-a-process/add-process-components).
- **Paso 3**: [Configurar los componentes del proceso](./design-a-process/configure-components).
- **Paso 4**: [Integrar el proceso en una interfaz de usuario](./design-a-process/integrate-process-into-ui).
