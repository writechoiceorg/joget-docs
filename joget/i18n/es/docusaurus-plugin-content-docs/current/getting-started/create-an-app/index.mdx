Después de cubrir los aspectos principales de la solución Joget e instalarla en tu sistema, es hora de crear una aplicación. Esta sección presenta las fases esenciales para crear tu primera aplicación en Joget.

Diseñaremos una aplicación de Solicitud de Viaje para ilustrar esta guía. Cada página presenta los pasos básicos para crear cada parte de esta aplicación. Sigue los pasos a continuación para tener la aplicación completa funcionando en solo unos minutos:

- **Paso 1**: [Diseñar una nueva aplicación](./create-an-app/design-new-app).
- **Paso 2**: [Definir un formulario](./create-an-app/design-a-form).
- **Paso 3**: [Crear una lista](./create-an-app/design-a-list).
- **Paso 4**: [Diseñar una nueva interfaz de usuario](./create-an-app/design-a-ui).
- **Paso 5**: [Establecer un proceso](./create-an-app/design-a-process).
