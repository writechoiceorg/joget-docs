Es hora de crear una lista después de crear la aplicación y el formulario. Utilizarás listas para mostrar datos a los usuarios. En este caso, la lista mostrará los datos del formulario de Solicitud de Viaje. Sigue los pasos en esta página para crear tu primera lista.

- **Paso 1**: Accede a la página del **compositor de aplicaciones**.
- **Paso 2**: Crea una nueva lista utilizando el botón **+** en la columna del **Constructor de Listas**.
- **Paso 3**: Ahora ingresarás los detalles de la lista, que incluyen:

<table>
  <thead>
    <tr>
      <th>Campo</th>
      <th>Descripción</th>
      <th>Valor para este ejemplo</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>ID</td>
      <td>Un identificador único.</td>
      <td>listaSolicitudes</td>
    </tr>
    <tr>
      <td>Nombre</td>
      <td>Un nombre descriptivo único.</td>
      <td>Lista de Solicitudes de Viaje</td>
    </tr>
    <tr>
      <td>Descripción</td>
      <td>Describe el propósito de la lista.</td>
      <td>Lista para presentar los datos adquiridos por el formulario de Solicitud de Viaje.</td>
    </tr>
  </tbody>
</table>

- **Paso 4**: Haz clic en **Guardar** para crear la lista. Como resultado, se lanzará el **Constructor de Listas**.
- **Paso 5**: Selecciona la fuente de datos. Las opciones disponibles están relacionadas con todas las fuentes de datos disponibles en Joget. Para este ejemplo, selecciona **Datos del Formulario**. A continuación, elige la opción **Seleccionar Datos (Datos del Formulario)**, que aparece justo encima del selector.
- **Paso 6**: Elige un formulario. Para este ejemplo, selecciona el **Formulario de Solicitud de Viaje**.
- **Paso 7**: Diseña tu lista. Primero, haz clic en el botón **Diseñar** en la parte superior de la página. Todas las columnas disponibles de la fuente de datos se enumeran a la izquierda. Arrastra y suelta las columnas deseadas para diseñar tu lista. Para este ejemplo, arrastra y suelta las siguientes columnas:

  - Nombre
  - Fecha de Viaje (Desde)
  - Fecha de Viaje (Hasta)
  - Motivo
  - Creado Por

:::info Editar columnas
Seleccionar cada columna de la lista te permite acceder a varias configuraciones que cambiarán la apariencia y el comportamiento de la lista.
:::

- **Paso 8**: Haz clic en **Vista Previa** para verificar cómo se ve la lista.
- **Paso 9**: Haz clic en **Guardar** cuando termines de crear la lista.
