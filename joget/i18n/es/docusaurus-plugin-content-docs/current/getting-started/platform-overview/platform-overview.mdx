Joget ofrece varias soluciones predefinidas para agilizar tu trabajo. En esta sección, presentamos los elementos esenciales disponibles en la plataforma Joget y las definiciones de importación. Utiliza los siguientes enlaces para navegar por las páginas de esta sección.

- [Aplicaciones y el Centro de Aplicaciones](./app-center): Aprende cómo acceder y encontrar todas las aplicaciones existentes en el portal de Joget. Además, aprenderás cómo utilizar el Mercado de Joget para adquirir aplicaciones de otros usuarios.
- [Interfaces de Usuario, Formularios, Listas y Procesos](./uis-forms-lists-process): Aquí encontrarás explicaciones sobre los principales componentes de las aplicaciones de Joget.
- [Barra del Administrador](./administrator-bar): Esta página presenta las funcionalidades disponibles en la Barra del Administrador, que es esencial para navegar por todas las funcionalidades de Joget.
- [Generador de Aplicaciones](./app-generator): El Generador de Aplicaciones es una solución de Joget para acelerar la creación de tus aplicaciones, proporcionando varias configuraciones predefinidas para crear nuevas aplicaciones rápidamente.
