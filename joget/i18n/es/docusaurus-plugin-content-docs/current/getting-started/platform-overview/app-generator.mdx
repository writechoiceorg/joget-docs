La función del Generador de Aplicaciones hizo su debut en la versión 4. Esta poderosa herramienta te ayudará a agilizar la creación de aplicaciones de flujo de trabajo de Joget, proporcionando un enfoque ágil tanto para la prototipación rápida como para el desarrollo completo, todo ello manteniendo una perspectiva centrada en el usuario.

El Generador de Aplicaciones es perfecto para impulsar el desarrollo de aplicaciones y facilitar las interacciones iniciales con los usuarios finales. Con sus capacidades, puedes dar forma automáticamente a una aplicación con listas, funcionalidad CRUD (Crear-Leer-Actualizar-Eliminar), un proceso de aprobación versátil y el elegante tema Twitter Bootstrap.

Para generar una nueva aplicación, primero debes definir sus componentes. Por lo tanto, te sugerimos que sigas la guía [Crear una aplicación](./create-an-app) primero.

## Uso del Generador de Aplicaciones

Para comprender el potencial del Generador de Aplicaciones, presentamos una guía sobre cómo crear una aplicación rápidamente a continuación.

1. **Diseña tu formulario**: Crea y diseña tu formulario según tus requisitos. Una vez que estés satisfecho, recuerda guardar el formulario. Si no estás familiarizado con la creación de formularios, consulta la guía [Crear una Aplicación](./create-an-app).
2. **Iniciar la generación de la aplicación**: Haz clic en **Generar Aplicación** en la esquina superior derecha del Constructor de Formularios.
3. **Navegar por el cuadro de diálogo del Generador de Aplicaciones**: Aparecerá un cuadro de diálogo que ofrece varias opciones para configurar tu aplicación según tus necesidades:

   - **Generar datalist**: Para mostrar datos.
   - **Generar CRUD**: Habilitar acciones de Crear-Leer-Actualizar-Eliminar.
   - **Incorporar proceso de aprobación**: Integración fluida de flujos de trabajo de aprobación.
   - **Incorporar proceso de asignación**: Simplificación de asignaciones de tareas.
   - **Generación avanzada**: Ajustar opciones para una salida personalizada.

4. **Genera tu aplicación**: Después de configurar las opciones, haz clic en **Generar**.
5. **Observa la transformación**: Actualiza tu página de diseño de aplicaciones después de que concluya el proceso de generación. Verás una serie de formularios generados automáticamente, listas, componentes de interfaz de usuario y procesos disponibles en el Compositor de Aplicaciones.
6. **Utiliza tu aplicación**: La nueva aplicación se publica automáticamente. Estará disponible en el Centro de Aplicaciones, donde puedes lanzarla y empezar a usarla.
