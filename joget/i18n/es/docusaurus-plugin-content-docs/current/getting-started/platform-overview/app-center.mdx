En el ecosistema de Joget, encontrarás varias aplicaciones que van desde las simples basadas en bases de datos hasta flujos de trabajo de aprobación más complejos y soluciones integrales como los sistemas de Gestión de Relaciones con el Cliente (CRM). Las aplicaciones son aplicaciones web diseñadas para ejecutarse en la plataforma Joget. Estas aplicaciones integran de manera fluida la gestión de flujos de trabajo, la administración de datos y capacidades de informes sólidos, potenciando tu viaje de transformación digital.

Tienes la libertad de personalizar o crear tu propia versión del Centro de Aplicaciones. Consulta la [guía de Personalización del Centro de Aplicaciones](https://dev.joget.org/community/display/DX8/Customizing+App+Center) para obtener información detallada sobre la personalización del Centro de Aplicaciones.

### Paso 1 - Acceso al Centro de Aplicaciones

El Centro de Aplicaciones es la página de inicio predeterminada dentro de Joget DX, que muestra una cuadrícula de aplicaciones publicadas diseñadas para mejorar tu eficiencia e innovación. Para acceder al Centro de Aplicaciones, utiliza un navegador web para navegar hasta tu servidor Joget. Puedes acceder a él utilizando el siguiente enlace [http://localhost:8080/jw](http://localhost:8080/jw). Si has utilizado un host o puerto diferente durante la instalación, cambia la URL.

### Paso 2 - Iniciar Sesión

Utiliza el botón **Iniciar Sesión** a la izquierda para acceder a la página de inicio de sesión y entrar en la plataforma como usuario. La experiencia de cada usuario está adaptada, lo que otorga acceso a aplicaciones distintas según los permisos individuales.

## Instalar Aplicaciones desde el Mercado Joget

El Mercado Joget proporciona una forma sencilla de integrar nuevas aplicaciones en tu instancia de Joget, expandiendo tus herramientas. Puedes acceder al Mercado Joget dentro del ámbito del Centro de Aplicaciones seleccionando **Descargar desde el Mercado**.

<Images
  imgURL="https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-08-28/3f40438d-bb65-456c-8c20-4bccc0921dd4/ascreenshot.jpeg?tl_px=604,95&br_px=1679,696&force_format=png&wat_scale=95&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=502,265"
  wSize="75%"
/>

:::info Acceso al Mercado Joget
No necesitas una cuenta en el Mercado Joget para acceder a él. Sin embargo, deberás registrarte para obtener una cuenta por separado en el Mercado Joget para poder descargar e instalar aplicaciones.
:::

Después de acceder al Mercado Joget, puedes buscar aplicaciones que satisfagan tus necesidades y seleccionarlas para obtener información adicional.

  imgURL="https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-08-28/974b27ec-5c7c-4357-b4c3-383d1745746f/ascreenshot.jpeg?tl_px=102,42&br_px=1177,643&force_format=png&wat_scale=95&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=502,265"

## Templates

**Templates** es una nueva función introducida en Joget DX 8 que permite a los diseñadores de aplicaciones crear plantillas para aplicaciones existentes modificando los nombres de las entidades en el diseño de la aplicación. Para utilizar la solución de plantillas, sigue los pasos a continuación:

1. Desde el Centro de Aplicaciones, accede a las plantillas seleccionando la opción **Plantillas**.

<Images
  imgURL="https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-08-28/7c9fc60d-85bb-4245-a50f-e2fa4871c9e7/ascreenshot.jpeg?tl_px=0,201&br_px=859,682&force_format=png&width=860&wat_scale=76&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=44,212"
  wSize="75%"
/>

2. Coloca el cursor sobre cualquier plantilla para revelar las opciones **Usar la Plantilla** y **Más Información**.

<Images
  imgURL="https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-08-28/46553575-81f8-46cc-aff8-20e2b9c8f4ff/ascreenshot.jpeg?tl_px=0,230&br_px=859,711&force_format=png&width=860&wat_scale=76&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=321,212"
