Joget ofrece componentes estándar para generar aplicaciones complejas, incluyendo interfaces de usuario (UI), formularios, listas y procesos. En esta página encontrarás una descripción básica de cada uno de estos componentes. Aprenderás cómo crearlos y utilizarlos en la página [Crear una Aplicación](./create-an-app).

## Interfaces de Usuario (UI): Donde la funcionalidad se encuentra con la experiencia del usuario

En el corazón de cada aplicación se encuentra la UI, la puerta de entrada a través de la cual los usuarios acceden a sus funciones. Una aplicación puede constar de una o varias UI, cada una con su propio tema único. Estas UI constan de páginas de menú que albergan una serie de elementos, como formularios, listas, informes, gráficos e incluso contenido HTML. La UI es tu lienzo para ofrecer una experiencia de usuario excepcional.

## Formularios: Capturando información de manera fluida

Los formularios son herramientas esenciales para recopilar y presentar datos. Vienen en varios formatos, desde campos de texto básicos hasta cuadrículas de formularios avanzados, lo que facilita la recopilación eficiente de información. Nuestra selección de elementos de formulario te proporciona la herramienta adecuada para cualquier tarea, ya sea una simple entrada de texto o una disposición de datos compleja.

## Listas: Información en forma de tabla

Las listas son hojas de cálculo organizadas con superpoderes que hacen que la visualización de datos sea agradable. Estas tablas ofrecen una gran cantidad de funciones, como paginación, ordenación, filtrado y exportación. Transforman los datos en bruto en información significativa manteniendo una apariencia estructurada.

## Procesos: Guiando el flujo de trabajo

Cuando utilizas Joget, un proceso es una serie de pasos conectados que trabajan juntos para lograr un objetivo o tarea específica dentro de una aplicación. Los procesos son esenciales para determinar el flujo de tareas, datos e interacciones dentro de la aplicación. Esto puede incluir tareas como enrutamiento y trabajo con sistemas externos, como el envío de correos electrónicos o la actualización de bases de datos.
